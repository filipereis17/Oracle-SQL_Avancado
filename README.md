# Curso Oracle SQL Especialista: do Básico ao Avançado
[![Curso Oracle SQL Avançado](https://img-c.udemycdn.com/course/750x422/3411880_4e40_3.jpg)](https://www.udemy.com/course/oracle-sql-especialista-do-basico-ao-avancado-completo/)

Este repositório contém o material de estudo para o curso avançado de Oracle SQL. O curso abrange 34 seções, das quais as 21 primeiras já foram estudadas no Curso Banco de Dados Oracle Completo: SQL, PL/SQL e Modelagem.

## Seções do Curso

**Seção 22 - Objetos do Banco de Dados**
   - Aprendi sobre os diferentes tipos de objetos presentes em um banco de dados Oracle e como trabalhar com eles.

**Seção 23 - Visões do Dicionário de Dados**
   - Explorei o dicionário de dados do Oracle e descobri como as visões do dicionário podem ser utilizadas para consultar informações sobre o banco de dados.

**Seção 24 - Índices**
   - Aprofundei-me no uso de índices para otimizar consultas e melhorar o desempenho do banco de dados.

**Seção 25 - Manipulação de Dados Avançada**
   - Aprendi técnicas avançadas para manipulação de dados, incluindo operações DML e DDL mais complexas.

**Seção 26 - Flashback**
   - Explorei o recurso de flashback do Oracle, que permitiu recuperar dados de forma rápida e eficiente.

**Seção 27 - Consultas Hierárquicas**
   - Dominei as consultas hierárquicas para lidar com dados organizados em estruturas de árvore.

**Seção 28 - Expressões Regulares**
   - Aprendi a usar expressões regulares para buscar e manipular padrões de texto dentro do banco de dados.

**Seção 29 - Tabelas Temporárias**
   - Descobri como usar tabelas temporárias para armazenar dados temporários de forma eficiente.

**Seção 30 - Globalização Tratamento de Datas e Time Zones**
   - Explorei como lidar com diferentes formatos de data, fusos horários e internacionalização no Oracle.

**Seção 31 - Subconsultas**
   - Aprofundei-me no uso de subconsultas para realizar consultas mais complexas e eficientes.

**Seção 32 - Gerando Scripts SQL Dinâmicos**
   - Aprendi a criar scripts SQL dinâmicos para automatizar tarefas e gerar consultas personalizadas.

**Seção 33 - Aperfeiçoando a Cláusula Group By**
   - Descobri técnicas avançadas para utilizar a cláusula GROUP BY de forma mais eficiente e poderosa.

**Seção 34 - Visões Materializadas**
   - Explorei as visões materializadas e como elas podem ser usadas para melhorar o desempenho de consultas complexas.
